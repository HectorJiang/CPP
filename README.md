服务器端：使用QTcpServer创建服务器，获取和客户端通信的套接字用于通信
QTcpServer类提供了一个基于TCP的服务器，通过该类可以快速建立TCP服务器，并接受客户端的连接请求；
listen监听地址，设置监听之后，每当检测到客户端发来连接请求，将会发送信号newConnection，在其中调用nextPendingConnection获取和客户端通信的套接字；
